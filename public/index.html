01 — Comment jouer

Quatre
étapes.

ZapPlay transforme n'importe quel écran en salle de jeu. Le host affiche la partie, chaque joueur rejoint depuis son téléphone avec un simple code. Pas de compte, pas d'install — juste du fun.

01
Ouvre une room
Un clic depuis n'importe quel navigateur — TV, ordi, tablette. Tu reçois un code à 4 lettres et un QR code.
+
02
Invite la bande
Les joueurs scannent le QR ou tapent le code sur leur mobile. Leur téléphone devient une manette.
+
03
Choisis le jeu
Quiz de culture, imposteur, dessin, débat mouvant… 20 modes différents pour varier les plaisirs.
+
04
Joue, rigole, recommence
Parties courtes (5–15 min). Les scores persistent, les titres se débloquent, la soirée s'écrit toute seule.
