ZapPlay · Loader universel

Loader sur
toutes les pages

Un seul fichier à inclure dans chaque page de jeu pour afficher l'écran de chargement identique au menu principal.

Intégration — 1 ligne

Copie zp-page-loader.js dans ton dossier public/, puis ajoute cette balise juste avant </body> dans chaque page :

<!-- ↓ Loader ZapPlay universel — coller avant </body> -->
<script src="zp-page-loader.js"></script>
1
Mode automatique auto
Le loader disparaît seul après le chargement de la page. Rien d'autre à faire.
2
Mode WebSocket ws
Pour les pages avec WebSocket, appelle window.ZPLoader.hide() dans ws.onopen pour que le loader disparaisse exactement quand la connexion est établie :
// Dans ton ws.onopen existant :
ws.onopen = () => {
  // ... ton code existant ...
  document.getElementById('dot').classList.add('ok');
  // ↓ Ajouter cette ligne
  window.ZPLoader?.hide();
};
